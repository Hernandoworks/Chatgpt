# Validators

Validation scripts and configurations used by the Phoenix AI Software Factory.

## Planned Validators

- `format.ts` — Prettier formatting check
- `lint.ts` — ESLint validation
- `typecheck.ts` — TypeScript type checking
- `tests.ts` — Test runner
- `security.ts` — Security scan (npm audit, code scanning)
- `schema.ts` — Prompt template schema validation
