# AI Software Factory — Prompt Library

This directory contains prompt templates for the Phoenix AI Software Factory.

Each prompt template is a markdown file that:

- Defines the role and objective
- Lists inputs required from the orchestrator
- Specifies the expected output format
- References relevant agents, standards, and templates

## Prompt Index

| Prompt                  | Description                                       | Agent                  |
| ----------------------- | ------------------------------------------------- | ---------------------- |
| `build-web-app.md`      | Build a full web application from a specification | Repository Engineer    |
| `build-ui-component.md` | Build a single UI component                       | UI Engineer            |
| `build-dashboard.md`    | Build a data dashboard                            | UI Engineer            |
| `build-api.md`          | Build an API endpoint or service                  | API Engineer           |
| `build-sql.md`          | Generate SQL schema, queries, or migrations       | API Engineer           |
| `build-sheet.md`        | Build a Google Sheets workbook                    | API Engineer           |
| `build-script.md`       | Build a Google Apps Script project                | API Engineer           |
| `review-code.md`        | Review code against project standards             | Solution Architect     |
| `refactor-code.md`      | Refactor code to improve quality                  | Repository Engineer    |
| `generate-docs.md`      | Generate documentation                            | Documentation Engineer |
